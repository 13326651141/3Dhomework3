写一个简单的总结，总结游戏对象的使用<br/>

**游戏对象：**
空对象，摄像机，光线，3D物体，声音，UI基于事件的new UI系统和粒子系统与特效

* 空对象（Empty）：虽然不可见，但却是最常用的对象之一
* 摄像机（Camara）：用眼观察整个游戏的唯一渠道
* 光线（Light）：游戏世界的氛围构造的重要组成部分
* 3D物体 ：3D游戏中的重要组成部分，一切场景的构造都要靠物体来实现
* 声音（Audio）：游戏中的声音来源
